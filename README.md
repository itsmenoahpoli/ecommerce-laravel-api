## Ecommerce-API

Made with Laravel 8.83.5

### Installation

```bash

git clone https://github.com/itsmenoahpoli/ecommerce-laravel-api/

cd ecommerce-laravel-api

git checkout develop

git pull

composer install

php artisan key:generate

php artisan migrate

php artisan db:seed

php artisan serve

```
