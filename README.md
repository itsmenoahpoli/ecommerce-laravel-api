## Ecommerce-API

Made with Laravel 8.83.5

### Installation

```bash

git clone https://bitbucket.org/itsmenoahpoli/ecommerce-laravel-api/src/master/

cd ecommerce-laravel-api

composer install

php artisan key:generate

php artisan migrate

php artisan db:seed

php artisan serve

```
